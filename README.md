## Demo
https://ashley-yu.github.io/Own_Space/

## 簡介
練習以 Vue.js 實作購物網站，功能區分為前、後台：
* 前台
  * 首頁 - 圖片輪播簡介、商品導覽
  * 商品列表 - 切換分類、加入購物車、加入按讚好物
  * 商品詳情 - 加入購物車、類似商品
  * 按讚好物 - 加入購物車
  * 購物車結帳 - 優惠券套用、訂單流程
* 後台
  * 管理者登入
  * 商品管理
  * 訂單管理
  * 優惠券管理
